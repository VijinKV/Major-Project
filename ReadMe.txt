Steps to run the project:-

Step 1) Install all the necessary files and library:
	Python,
	Python sub Library,
	Chromedriver.exe,
	Selenium Library

Step 2) Run DataScrapping.py after the execution results will be stored in the current location
 	for further execution.

Step 3) Execute Correlator.py using python which will calculate and store the results in CSV
 	format and the graphs will be stored in png format.
